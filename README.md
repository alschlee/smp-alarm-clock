# Alarm Clock

p5.js를 이용한 인터랙티브 알람 시계 웹 애플리케이션  
시/분/초를 입력하여 원하는 시간에 알람을 설정하고, 커스텀 사운드로 알림을 받을 수 있습니다.

## 주요 기능

- 시/분/초 단위로 알람 시간 설정
- 여러 개의 알람 동시 등록 및 관리
- 실시간 현재 시각 표시
- 알람 시간이 되면 자동 사운드 재생
- 개별 알람 삭제 및 정지 기능
- 커스텀 알람 사운드 지원

## 기술 스택

- p5.js - 그래픽 및 UI 처리
- p5.sound.js - 오디오 파일 재생

## 사용 방법

1. **알람 추가하기**
   - 시/분/초 입력란에 원하는 시간을 입력합니다
   - `+ 추가` 버튼을 클릭하여 알람을 등록합니다
   - 여러 개의 알람을 추가할 수 있습니다

2. **알람 관리하기**
   - 등록된 알람 목록에서 각 알람의 시간을 확인할 수 있습니다
   - `삭제` 버튼으로 알람을 제거할 수 있습니다
   - 알람이 울릴 때 `멈춤` 버튼으로 사운드를 정지할 수 있습니다

3. **알람 울림**
   - 설정된 시간이 되면 자동으로 알람 사운드가 재생됩니다
   - 알람이 울리는 동안 `멈춤` 버튼이 표시됩니다

## 입력 형식

- **시(Hour)**: 0 ~ 23 (24시간 형식)
- **분(Minute)**: 0 ~ 59
- **초(Second)**: 0 ~ 59

예시: 오후 3시 30분 → 15시 30분 00초

## 커스터마이징

### 알람 사운드 변경

```javascript
// sketch.js 상단에서
let soundFileName = 'your-sound.mp3';  // 원하는 사운드 파일명으로 변경
```

### 캔버스 크기 조절

```javascript
// setup() 함수에서
createCanvas(500, 750);  // 너비, 높이
```

### 색상 테마 변경

```javascript
// draw() 함수에서
background(250, 248, 252);  // 배경색 (R, G, B)

// 입력란 배경색
inputHour.style('background-color', '#EDE7F6');   // 시 - 연보라
inputMinute.style('background-color', '#D1C4E9'); // 분 - 중간 보라
inputSecond.style('background-color', '#B39DDB'); // 초 - 진보라
```

## 주의사항

- 알람 사운드 파일이 같은 디렉토리에 있어야 정상 작동합니다
- 여러 알람이 동시에 울릴 경우 각각의 사운드가 중복 재생됩니다
